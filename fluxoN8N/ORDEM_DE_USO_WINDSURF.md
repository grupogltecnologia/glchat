# Ordem recomendada para trabalhar no Windsurf

1. Abra `04_prompts_windsurf/00_PROMPT_MASTER.md` e peça apenas análise/plano.
2. Depois execute `01_FRONTEND_HTML_PHP.md`.
3. Depois `03_BANCO_MYSQL.md`.
4. Depois `02_BACKEND_PHP_MYSQL.md`.
5. Depois `04_INTEGRACOES.md`.
6. Depois `05_AGENTES_IA_ANALISE.md`.
7. Por último `06_TESTES_VALIDACAO.md`.

## Como evitar confusão
- Não peça para o Windsurf ler o JSON inteiro primeiro.
- Use primeiro o `MAPA_TECNICO.md`.
- Quando precisar de regra específica, entregue o arquivo separado de `03_json_por_modulo/`.
- O JSON original está na pasta `00_original_json/`, mas deve ser usado só para conferência.
